# System Use Cases

This document lists the main **functional capabilities** of the Pharmacy Assistant system.

Use cases represent the actions the system can perform.

They are organized by module.

---

# Catalog Module

Manage promotional offers available for recommendation.

Use cases:

- Create Offer
- List Offers
- Update Offer
- Disable Offer

---

# Knowledge Module

Provide curated pharmacy knowledge used by the assistant.

Use cases:

- Create FAQ
- List FAQs
- Update FAQ
- Disable FAQ

---

# Inquiries Module

Track user questions and assistant responses.

Use cases:

- Register Inquiry
- List Inquiries

---

# Assistant Module

Generate AI-assisted recommendations.

Use cases:

- Build Assistant Context
- Recommend Offers
- Generate Assistant Response
